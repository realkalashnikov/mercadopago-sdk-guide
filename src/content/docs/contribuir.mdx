Este guia é um projeto de código aberto mantido de desenvolvedor para desenvolvedor. Se você quer ajudar a economizar o tempo de outros devs que estão quebrando a cabeça com o Mercado Pago, sua contribuição é muito bem-vinda!

---

## O que precisamos de ajuda?

Atualmente, nosso maior objetivo é preencher a pasta `/examples` com códigos funcionais completos para todas as linguagens. Temos exemplos rodando em **Node.js (Express)** e **Python (FastAPI)**, mas precisamos de ajuda para completar os templates de:

* **PHP (Laravel)**
* **Go (Fiber)**
* **.NET (C#)**
* **Java (Spring Boot)**
* **Ruby (Rails)**

Se você domina alguma dessas tecnologias, considere enviar um Pull Request!

---

## Como ajudar?

Todo o fluxo de contribuição, comandos locais para rodar a documentação e os padrões de código exigidos (como o uso da SDK v2, webhooks rápidos e validação HMAC) estão detalhados no arquivo de contribuição no GitHub:

👉 **[Ler Diretrizes de Contribuição no GitHub](https://github.com/realkalashnikov/portfolio)**

Obrigado por ajudar a tornar a integração de pagamentos no Brasil menos dolorosa!
